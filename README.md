# LaraSpeed Support

Welcome to the official support repository for [LaraSpeed](https://laraspeed.dev).

## How to Get Help

**Before opening an issue, please:**
1. Check the [Documentation](https://laraspeed.dev/docs)
2. Search [existing issues](https://github.com/jana-gocanin/laraspeed-support/issues) — your question may already be answered

## Opening an Issue

- **Bug reports:** Use the Bug Report template
- **Feature requests:** Use the Feature Request template
- **Questions:** Use the Question template
