## Features

- **[Authentication & Authorization](https://laraspeed.dev/docs/authentication)** — Login, registration, email verification, 2FA, social login
- **[Subscription Billing](https://laraspeed.dev/docs/billing)** — Stripe & Lemon Squeezy integration with plans, trials, and invoices
- **[Admin Panel](https://laraspeed.dev/docs/admin-panel)** — Filament-powered dashboard for managing users, plans, and settings
- **[Team Management](https://laraspeed.dev/docs/teams)** — Multi-tenancy with roles, invitations, and permissions
- **[User Dashboard](https://laraspeed.dev/docs/dashboard)** — Profile management, billing portal, and activity overview
- **[Blog & SEO](https://laraspeed.dev/docs/blog)** — Built-in blog with meta tags, sitemap, and Open Graph support
- **[Email Templates](https://laraspeed.dev/docs/emails)** — Pre-designed transactional emails ready to customize
- **[API Ready](https://laraspeed.dev/docs/api)** — Token-based API authentication with Sanctum

## Why LaraSpeed?

Skip weeks of boilerplate setup. LaraSpeed handles the repetitive SaaS infrastructure so you can focus on building what makes your product unique.

- Production-tested codebase
- Clean, well-documented code
- Regular updates and new features
- Active community and support

## Built With

LaraSpeed is built on top of trusted open-source technologies:

- [Laravel](https://laravel.com) — PHP web framework
- [Filament](https://filamentphp.com) — Admin panel and form builder
- [Livewire](https://livewire.laravel.com) — Reactive components
- [Tailwind CSS](https://tailwindcss.com) — Utility-first CSS framework
- [Stripe](https://stripe.com) / [Lemon Squeezy](https://lemonsqueezy.com) — Payment processing
